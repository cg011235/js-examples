Interesting facts about JS

The script tag can embed JS code in a HTML document

Larger code can be implemented in JS script and script tag can take path to it.

JS works from frontend like ReactJS to backend like NodeJS.

The script language= tag does not makes sense as by default JS is the language

The script type= tag is no more required

The semicolons are preferred but not compulsory, if multiple statements are on same line semicolon is needed
